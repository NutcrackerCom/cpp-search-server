# C++ Search Server

Консольный поисковый сервер по ключевым словам.

## Возможности

* добавление и индексация документов;
* поиск по релевантности TF-IDF;
* учёт стоп-слов и минус-слов;
* сортировка результатов по релевантности и рейтингу;
* фильтрация документов по статусу;
* пагинация результатов;
* подсчёт запросов без результатов.

## Сборка и запуск

```bash
git clone https://github.com/NutcrackerCom/cpp-search-server.git
cd cpp-search-server

g++ -std=c++17 search-server/*.cpp -o search_server
./search_server
```

## Стек

* C++17;
* стандартная библиотека C++;
